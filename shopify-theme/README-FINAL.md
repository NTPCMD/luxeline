## ğŸ› ï¸ Troubleshooting

### **Logo not showing?**
- Make sure you uploaded the logo in Theme Settings â†’ Logo & Branding
- Try clearing your browser cache
- Logo should be PNG or JPG, max 2MB

### **Menu not appearing?**
- Verify you created a menu named "Main menu" in Navigation
- Assign the menu in Header section settings (in theme editor)

### **Products not showing on homepage?**
- Make sure products are published and active
- Check that collections have products assigned
- Verify sections are configured to display correct collections

### **Mobile menu not working?**
- Clear browser cache and refresh
- Check that theme.js is loaded (view page source, search for `theme.js`)

### **Colors look different?**
- Colors may appear different on various screens
- Double-check color codes in Theme Settings
- Some browsers may have night mode enabled
