# LuxeLine Shopify Theme

## Install
1. Zip the `shopify-theme` folder.
2. In Shopify Admin → Online Store → Themes, upload the zip file.
3. Publish the theme.

## Templates
- **Home:** `index.json`
- **About:** `page.about.json`
- **FAQ:** `page.faq.json`
- **Contact:** `page.contact.json`
- **Launch:** `page.launch.json`

Assign page templates in Shopify Admin → Pages → Theme template.

## Launch date
Open the **Launch countdown** section in the Theme Editor and set the **Launch date (ISO format)** setting (example: `2026-06-01T00:00:00+08:00`). If the date is empty, the countdown hides and "Coming Soon" is shown instead.
